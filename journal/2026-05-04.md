## AlterEgo reverse engineering

Watched the AlterEgo commercial demo (alterego.io). Key observations:

**What the demo actually shows:**
- Free-form question answering about physical objects ("what is this postcard?")
- Two-person silent conversation ("where do you want to get lunch?")
- Reminder creation, Mandarin translation
- "Tiny cameras built in" — forward-facing AR cameras on the wearable

**The camera is for AR/vision (pointing at objects), not for speech recognition.**
Speech recognition is still pure EMG. The camera feature and the speech feature
are separate systems bolted together.

**Why it looks like magic:**
1. Rigid wearable pinches tightly around jaw/ear — guarantees identical electrode
   placement every session. This is the actual breakthrough, not an algorithm.
   Consistent placement means every training session stacks cleanly, no session
   mismatch, no placement variance.
2. Single user, extensively trained on themselves over a long period
3. Closed vocab for the actual EMG recognition — the "free form" conversation
   is almost certainly a fixed vocab exchange, editing hides the constraints
4. LLM post-processing on top — 85% EMG accuracy becomes ~98% perceived
   accuracy because the language model fills gaps from context
5. Heavy video editing — staged exchanges, cherry-picked takes

**No technological breakthrough required to replicate this.** The ingredients
are all known and available:
- Rigid TPU 3D-printed jaw wearable for consistent placement (v2 hardware)
- Single-user training data collected obsessively
- Modern transformer architecture (2026 models >> 2018 CNN)
- LLM post-processing layer

**The subvocal gap** is the only genuinely hard part — getting the model to
work with zero visible mouth movement. The two-model cascade plan (overt
first, distill to subvocal with paired data) is the right approach. AlterEgo
likely solved this the same way: massive amounts of paired overt/subvocal data
from a single user over time.

## Form factor direction

Long-term wearable plan: rigid TPU 3D-printed piece that hugs jaw/ear area,
snap connectors embedded at fixed electrode positions. Put it on, electrodes
land in the same spot every time, no prep beyond a dab of gel.

Bilateral placement (both sides of head, same as AlterEgo) doubles channels
and improves robustness. Current OpenBCI clone supports 4 bipolar channels —
enough for bilateral masseter + bilateral orbicularis in bipolar mode.
